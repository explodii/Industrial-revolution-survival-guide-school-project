How's This Look?

JUUUST making some color palette changes, don't mind me!

(The old palette was so nice I actually want to use it for something, so I made a new one, lol...)

This style is mobile friendly, should look good on small screens, and has an option dark/light mode based on user browser preferences! (If you're on Firefox, you can even open up web developer tools with Ctrl+Shift+I and toggle the dark/light mode manually with the little sun and moon icons. Neat, right?)

Click on the tabs at the top to go to different pages and see the full behavior of this style. You can set whichever tab you want as the "highlighted" tab, but it's meant to be for the current page you're on in the nav/menu.
